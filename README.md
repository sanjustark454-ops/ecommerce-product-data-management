# E-commerce Product Data Management

## Project Overview
Python-based data analysis project built using Jupyter Notebook to manage and analyze e-commerce product data.

## Objectives
- Clean and preprocess product data
- Perform exploratory data analysis (EDA)
- Generate insights from product information

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## File
- E-commerce_Product_Management.ipynb
